# Cryptarena вҖ” UI Component Specification

## Design Language
- **Theme**: Dark cyberpunk with neon accents
- **Primary BG**: #050d1a (deep navy)
- **Text**: #d0eaff (cool white-blue)
- **Accent 1**: #00d4ff (cyan)
- **Accent 2**: #00ff88 (green вҖ” profit)
- **Danger**: #ff2d55 (red вҖ” loss)
- **Gold**: #ffd700 (gold вҖ” legendary/premium)
- **Font**: Inter (UI), monospace (numbers/data)
- **Corners**: 8px radius standard, 16px for cards

## UI Animation Guidelines

1. **Transitions**: 300ms ease-out for panel slides
2. **Numbers**: Count-up animation for P&L changes
3. **Jackpot**: Slot-machine spin effect before reveal
4. **Stage transitions**: Full-screen flash in stage color
5. **Bot damage**: Screen shake + red vignette
6. **Bot profit**: Green pulse from bot outward
7. **Evolution**: Particle implosion вҶ’ explosion on upgrade
