# CoinBattle Saki — Character Archetype System

## Archetype Template

Each character is generated from this template:

```yaml
name: [Name]
name_ja: [Japanese name]
philosophy: [Core belief about trading/markets]
trading_style: [momentum | mean_reversion | high_frequency | event_driven | arbitrage | contrarian]
visual_motif: [Robot design concept]
primary_color: [Hex color]
weapon: [Signature weapon/tool]
myth_reference: [Mythological inspiration]
personality: [Character personality in one sentence]
battle_cry: [Japanese battle phrase]
region: [Home region]
rank: [S/A/B/C/D/E]
```

## Core Archetypes

### The Commander (指揮官)
- Calm, strategic, sees the whole battlefield
- Mean reversion / multi-factor
- Example: Saki

### The Berserker (狂戦士)
- All-in momentum, never retreats
- Pure trend following
- Example: Musashi

### The Assassin (暗殺者)
- Lightning speed, hit and run
- High frequency scalping
- Example: Raiko

### The Oracle (預言者)
- Patient, waits for perfect setup
- Event-driven, deep analysis
- Example: Sterling

### The Shapeshifter (変幻者)
- Adaptive, changes strategy mid-battle
- Regime-switching models
- Example: Long

### The Fortress (要塞)
- Maximum defense, compound growth
- Ultra-conservative risk management
- Example: Ori

### The Trickster (策士)
- Unpredictable, exploits opponent psychology
- Contrarian / adversarial
- Example: ZERO

### The Titan (巨人)
- Overwhelms with size and resources
- Large position accumulation
- Example: Malik

## Generation Rules

1. Every character must have a clear trading philosophy
2. Visual design must reflect the philosophy
3. Weapon/tool must be metaphorical for trading approach
4. Myth reference adds cultural depth
5. Battle cry in Japanese adds identity
6. No two characters should have identical strategies
7. Characters should feel like Street Fighter bosses / Saint Seiya warriors
