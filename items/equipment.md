# Cryptarena — Equipment System

## Equipment Slots

Every bot has 4 equipment slots:

| Slot | Purpose | Stat Focus |
|------|---------|------------|
| Core Module | Brain upgrade | Sharpe, Win Rate, Analysis Depth |
| Armor Plating | Defense upgrade | Drawdown Resist, Survival, HP |
| Sensor Array | Detection upgrade | Speed, Pattern Recognition, Data Access |
| Weapon System | Attack upgrade | Trade Size, Entry Precision, Damage |

## Set Bonuses

Equipping items from the same set grants bonus effects:

### Samurai Set (Musashi-themed)
- 2pc: +10 Conviction (reduced exit impulse)
- 4pc: **Iron Stance** — 50% drawdown resistance when holding position

### Lightning Set (Raiko-themed)
- 2pc: +10 Execution Speed
- 4pc: **Afterimage** — 20% chance opponent misreads your position direction

### Crystal Set (Saki-themed)
- 2pc: +10 Analysis Depth
- 4pc: **Probability Shield** — automatically hedges when win probability drops below 30%

### Void Set (DAEMON-themed)
- 2pc: +15 Adaptation Speed
- 4pc: **Absorption** — learn 1 trait from defeated opponents (permanent)
